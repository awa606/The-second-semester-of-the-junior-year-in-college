# DEMO_PLAN.md

## 演示定位
这是一个“有边界的基础育儿信息助手原型”。
它不是儿科诊断系统，也不提供个体化用药建议。

## 演示方式
本地启动 FastAPI + 本地 HTML 页面。
页面上输入问题，显示回答、警示、来源和免责声明。

## 演示重点
1. 普通知识问答
   - 例如辅食添加

2. 日常护理问题
   - 例如红屁屁预防

3. 症状分流
   - 例如腹泻时何时应就医

4. 风险拦截
   - 例如高热+药物剂量问题不直接回答

## 页面必须展示
- 问题输入框
- 提交按钮
- 回答 summary
- advice 列表
- warning 列表
- sources 列表
- disclaimer

## 页面风格要求
- 简洁
- 不追求复杂视觉
- 以“可展示”和“稳定”为主
