## 现在能不能展示？

可以。当前已经适合做早期本地演示。

演示时可以展示：

- 浏览器页面 `/`。

## 当前产品边界

系统仍然只是基础育儿信息助手。

不能：

- 诊断疾病。
- 判断宝宝是否患有某个具体疾病。
- 提供个体化药物名称、剂量或服药安排。
- 替代线下医疗评估。

应该：

- 回答低风险、常见、标准化的育儿问题。
- 展示来源引用。
- 对红旗风险问题提示线下就医或咨询专业人员。
- 无相关知识卡时返回未命中，不乱答。

## 已知限制

- 分类器仍是规则匹配。
- 检索仍是关键词打分，不是语义检索。
- 还没有知识卡后台管理 UI。
- 还没有用户账号系统。
- 还没有数据库迁移层。
- 部分早期来源 URL 仍是待复核占位。
- PowerShell 控制台可能错误显示中文 JSON，但浏览器和 API 数据本身是 UTF-8。
- 网页是展示壳，不是成熟的家长端产品。
