# 项目手册

这个文件夹存放 `parenting_kb_v1` 的持续维护手册。

当行为、数据、部署方式或产品范围发生变化时，需要同步更新这里的文档。目标很简单：下一个接手的人打开本目录后，能快速知道当前原型能做什么、最近改了什么、还差什么。

## 文件说明

- `PROJECT_STATUS.md`：当前演示状态、知识覆盖和已知限制。
- `ROADMAP.md`：从本地原型走向更完整产品的路线。
- `MINIPROGRAM_INTEGRATION.md`：如何把当前后端接入微信小程序。
- `DOCUMENTATION_RULES.md`：以后修改代码和知识库时如何记录文档。
