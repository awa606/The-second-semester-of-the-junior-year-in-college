# docs 目录说明

这个目录存放项目文档、版本记录、交接资料和审核资料。

## 文件说明

- `CHANGELOG.md`：每次实质修改都要记录。
- `RUN_AND_SHARE.md`：本地运行、局域网演示和公网分享方式。
- `manual/`：项目状态、路线图、微信小程序接入和文档规则。
- `archive/`：历史交接和阶段计划文件。
- `review/`：知识来源审核相关资料。

## 使用方式

接手项目时建议先读：

1. 根目录 `README.md`
2. `docs/CHANGELOG.md`
3. `docs/manual/PROJECT_STATUS.md`
4. `docs/manual/ROADMAP.md`

## 修改提醒

有实质修改时，至少更新 `CHANGELOG.md`。如果目录结构、运行方式、知识库或小程序接入方式变化，也要更新对应文档。
