# AGENTS.md

## 项目目标
本仓库用于完成第 5 周 CLIP 图片打标签实践项目中的任务 1：电商商品质检助手。

## 开发优先级
1. 保持 CLIP 零样本分类方案。
2. 先跑通后端接口与前端调用。
3. 再做标签优化与评测。
4. 最后补 README 和展示材料。

## 代码偏好
- Python 代码需包含中文注释。
- 尽量把模型加载和推理逻辑拆开。
- 避免把所有逻辑堆在一个脚本里。
- 若修改现有文件，优先保留原注释和原结构。

## 默认技术方案
- 模型：`openai/clip-vit-base-patch32`
- 推理：`transformers.pipeline("zero-shot-image-classification")`
- 后端：FastAPI
- 前端：原生 HTML + JS 或最小可运行静态页面
- 演示：局域网访问，A 电脑跑后端，B 电脑作为前端访问

## 接口规范
### GET /health
返回：
```json
{"status": "ok"}
```

### POST /predict
输入：表单文件 `file`

输出示例：
```json
{
  "top1_label": "完好商品",
  "top1_score": 0.893,
  "top3": [
    {"label": "完好商品", "score": 0.893},
    {"label": "外包装破损商品", "score": 0.072},
    {"label": "空盒商品", "score": 0.035}
  ],
  "advice": "✅ 商品完好，建议重新上架"
}
```

## 标签建议
优先尝试更具体的视觉描述，而不是泛化名词。
